# 超市进销管理系统

主要模块分为四块：

1. 文件处理模块：主要负责文件的读写
2. 窗口实现模块：文档界面和对话框界面的搭建，时间的显示，界面的挂载，界面风格的确立。
3. 主要功能模块：编写应用程序，控制控件之间底层的联系和对话框之间消息的传递。
4. 消息处理模块：处理Windows消息队列中的信息，根据信息产生对应行为。
